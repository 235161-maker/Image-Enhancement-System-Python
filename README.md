# Digital Image Processing - Image Enhancement System

## Project Overview
A comprehensive image enhancement application that integrates multiple image processing techniques to improve low-quality images.

## Features
- Image loading and analysis
- Sampling and quantization operations
- Geometric transformations (rotation, translation, shearing)
- Intensity transformations (negative, log, gamma)
- Histogram processing and equalization
- Complete enhancement pipeline
- Professional GUI interface
- PDF report generation

## Installation

### Prerequisites
- Python 3.8+
- pip package manager

### Setup
```bash
# Clone the repository
git clone [your-repo-link]

# Navigate to project directory
cd DIP-Image-Enhancement-System/code

# Install dependencies
pip install -r requirements.txt

# Run the application
python main.py
